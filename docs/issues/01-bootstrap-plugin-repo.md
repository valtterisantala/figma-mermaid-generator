# Issue 01 — Bootstrap plugin repo and build pipeline

## Purpose

Create the base Figma plugin repo and development setup.

## Scope

- add `manifest.json`
- add `package.json`
- add TypeScript config
- add build setup
- add linting and formatting
- add base folder structure for:
  - `main`
  - `ui`
  - `core`
  - `fixtures`
  - `tests`
- add minimal README with local setup instructions

## Deliverables

Suggested structure:

```text
src/
  main/
  ui/
  core/
  fixtures/
  tests/
docs/
```

## Definition of done

- plugin repo can be installed and built locally
- manifest is valid for a Figma plugin
- TypeScript compiles cleanly
- development folder structure is in place
- README explains local setup
